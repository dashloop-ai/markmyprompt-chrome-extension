# Icons

Place the following icon files in this directory:
- `icon16.png` (16x16 pixels)
- `icon32.png` (32x32 pixels)
- `icon48.png` (48x48 pixels)
- `icon128.png` (128x128 pixels)

These icons will be displayed in:
- Chrome toolbar (icon16, icon32)
- Extension management page (icon48, icon128)
- Chrome Web Store listing (icon128)

## Recommended Design
- Use the Promptean brand colors (purple gradient: #667eea to #764ba2)
- Include a lightning bolt or sync symbol
- Keep it simple and recognizable at small sizes

## Temporary Solution
For development, you can use simple colored squares or generate icons using online tools like:
- https://www.favicon-generator.org/
- https://realfavicongenerator.net/
